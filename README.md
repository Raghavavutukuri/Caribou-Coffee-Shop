# Caribou-Coffee-Shop
This is a food ordering website named as Caribou Coffee Shop, with this website registered customers can place food orders
> *Technologies used*

*Front End:*
HTML, 
CSS,
Java Script,
JQuery

*Back End:*
PHP

*Database language:*
SQL

*Local Hosting server:*
XAMPP

>Time Taken to Finish this project : 1 Week
